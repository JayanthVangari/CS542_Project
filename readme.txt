Instructions to compile and run program:

1. The program is an executable jar file 'LinkStateAlgo.jar'
   to run the jar file : java -jar LinkStateAlgo.jar

2. After running the jar file, the user is asked to give input file
   Give the complete path of the file with extension 
	for example : 'D:/j/semester/CS542/project/routercost.txt'

3. Then user is asked to choose from the options displayed on the menu
	
4. If the user want to view the forwarding table enter the command '1'

5. If the user want to find the least cost path between source node and destination node enter the command '2'

6. If the user want to exit enter '-1'.
